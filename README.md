# FantasticWorld

**FantasticWorld** 是一个基于 **Unreal Engine 5** 的开源第三人称动作冒险游戏项目，用于探索 **开放世界 + 技能驱动战斗 + 刷宝成长** 的游戏设计与实现方式。

> 当前仓库主要目标是完成一个可游玩的 Demo，用于验证核心玩法与系统架构。

---

## 项目特点

* 🎮 **武器即职业**：切换武器即可在战士 / 弓箭手 / 魔法师之间切换
* ⚔️ **技能驱动战斗**：以技能为核心，包含可连续释放的无消耗技能（类普攻）
* 🌍 **开放区域探索**：单张开放地图，包含敌人、宝箱与 Boss
* 🎒 **装备与刷宝系统**：装备随机词条与套装效果，驱动角色成长
* 🎨 **赛璐璐风格（Cel-Shaded）**：参考《塞尔达传说：旷野之息》

---

## 技术栈

* **引擎**：Unreal Engine 5
* **核心系统**：

  * Advanced Locomotion System (ALS)
  * Gameplay Ability System (GAS)
* **模式**：纯单机

---

## Demo 范围

* 单一开放区域（类似“初始台地”）
* 1–2 小时体验内容
* 一条引导式主线任务
* 最终 Boss 战
